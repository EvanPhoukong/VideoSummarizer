This tool analyzes video content and generates concise, structured abstract notes. 
Ideal for lectures, tutorials, and presentations, it extracts key concepts, summarizes discussions, and produces readable notes to save time and enhance learning.

NOTE: This tool requies an OpenAI API Key to be added to the system environment variables.
- Generate a key here: https://platform.openai.com/api-keys
- Directions to add API Key to System Environment: https://help.openai.com/en/articles/5112595-best-practices-for-api-key-safety
- A credit balance above $0 is REQUIRED. This model uses very little tokens (Average < $0.005 per run when summarizing one of the 3 preconfigured videos).
- To view the price of GPT 5 Nano (the model being used), refer to this link: https://openai.com/api/pricing/
